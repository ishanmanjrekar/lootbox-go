# Android Assets Folder

Please place your gacha game icon image in this folder:
- **Filename**: `icon.png`
- **Size**: `1024x1024` pixels (PNG)
- **Safe Zone**: Keep the main logo elements in the middle `512x512` pixel range so it won't get cut off on circular adaptive launcher icons.

Once you add the image here, we can use it to update both the in-game loading splash screen and the native Android launcher icons!
