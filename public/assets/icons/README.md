# Lootbox Go! - Icons & UI Assets Specification

This folder holds the UI icons and graphical assets for the interface. Please place the assets here in **transparent PNG format (`RGBA`)** matching the Cute Kawaii Pinata Party style (rounded shapes, soft chocolate-brown borders, warm tones).

---

## 1. Icon Specifications & Sizes

| Filename | Description | Dimensions | Location / Usage |
| :--- | :--- | :---: | :--- |
| **`energy.png`** | A yellow-orange cartoon flame icon with kawaii blush cheeks. | $128 \times 128\text{ px}$ (or $256 \times 256\text{ px}$) | Top HUD (Energy balance), Open Box button |
| **`pinata.png`** | A colorful candy-pinata head or body icon with happy eyes. | $128 \times 128\text{ px}$ (or $256 \times 256\text{ px}$) | Top HUD (Pinata balance), Store packages |
| **`xp_crown.png`** | A bubble-gum pink cartoon crown representing XP/Level. | $128 \times 128\text{ px}$ | XP Progress Bar (left-hand badge), Level Up screens |
| **`shopping_cart.png`** | A playful blue cartoon shopping cart or gift basket. | $128 \times 128\text{ px}$ | Top HUD (Store trigger button) |
| **`kofi_logo.png`** | A stylized cartoon coffee cup with a small pink/red heart on it. | $128 \times 128\text{ px}$ | Store Screen (Ko-fi tipping banner button) |
| **`box_locked.png`** | A grayed-out mystery box silhouette with a question mark. | $512 \times 512\text{ px}$ | Collection Drawer (Locked item placeholders) |

---

## 2. Style Requirements
- **Outlines:** Use soft chocolate-brown outlines (`#2b201d` or similar dark warm brown) rather than harsh black borders.
- **Transparency:** Alpha channel enabled (`RGBA`) with clean masking (no bounding boxes or white borders).
- **Aesthetic:** Marshmallow-like, squishy, and friendly cartoon style.
